Role

Action expression​

The role action manipulates a target's roles.

Possible arguments​

Add​

Defines a set of roles to add on the target.

This argument takes a role mention value.

ValueMax ValuesDescription
Role Mention15Example: @testrole (interpreted as <@&929204504572862537>)

Default listener behavior overrides​

When connected with the reaction listener, when a user removes the reaction they performed, ComfyAI also removes the role they gained. This is a convenience feature that can be disabled with the defaults=false argument on the reaction listener.

This prevents you from having to manually define removing the role in a separate when command.

Example​

when reaction emote= <:USA:840311365813010492> message= 934204818023653418 -> role add= @american

When the user reacts to a specific message with the USA flag emote, give them the role "American".

Remove​

Defines a set of roles to remove on the target.

This argument takes a role mention value.

ValueMax ValuesDescription
Role Mention15Example: @testrole (interpreted as <@&929204504572862537>)

Set​

Defines the roles to set on the target. All roles will be removed, and replaced with the defined set.

This argument takes a role mention value.

ValueMax ValuesDescription
Role Mention15Example: @testrole (interpreted as <@&929204504572862537>)

Target​

Defines which user will have their roles manipulated. By default the user triggering the listener is targetted.

This argument takes a user mention value.

ValueDescription
User MentionExample: @username (interpreted as <@929204504572862537>)
