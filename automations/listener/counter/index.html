Counter

Listener expression​

The counter listener executes when the counter reaches the defined count.

A counter has two parts, the count action and the counter listener. When a count action increments the count of a specific key to the set amount, the counter listener triggers an action.

Suggested use cases​

The counter is a useful listener that's great for community involvement and voting. Here's some ideas:

  • You could increment the count when users react to a meme with the star emote. Once the emote count has reached a number you choose you can use the counter listener to repost the popular message to a favorites channel.

  • Counter can be used for voting. When the specified count (defined on the count action, not the counter listener) the listener will declare the winner.

Possible arguments​

Key​

Defines the key name of the counter to be used in the count action.

This argument takes a string (raw text surrounded by "") value.

ValueDescription
StringA arbitrary name for your counter to be used in reference to an action count

Example​

when counter key= "my_key" -> message content="The count was reached!" channel= <#918390434848464907>

When the count is reached a message is sent in the specified channel.
